## 1. Welcome to Gold.Arch

### 1.1 What is Gold.Arch?

Gold.Arch Construction CRM is your all-in-one platform for managing construction supplier relationships, projects, deals, and quotes. Designed specifically for construction companies, Gold.Arch helps you:

- **Centralize Supplier Data** - Keep all your supplier information in one place
- **Track Projects** - Manage project lifecycle from planning to completion
- **Manage Deals** - Track negotiations and close deals faster
- **Handle Quotes** - Create, send, and track quotes effortlessly
- **Stay Organized** - Assign tasks and log activities

#### Step-by-Step Instructions:

1. **Visit the Website**
   - Open your browser and navigate to the Gold.Arch website
   - You'll see the landing page with supplier categories and features

2. **Click Sign Up**
   - Click the "Get Started" button in the hero section, OR
   - Click "Sign In" in the top navigation, then select "Sign Up"

3. **Enter Your Details**
   - Full Name: Your complete name
   - Email: Your work email address
   - Password: Choose a strong password (8+ characters recommended)

4. **Access Your Dashboard**
   - After successful registration, you'll be redirected to your dashboard
   - Explore the interface and start adding suppliers

### 5.4 Creating a Project

1. Navigate to **Projects** in the sidebar
2. Click **"+ New Project"**
3. Fill in project details:
   - Project name
   - Description
   - Start date / End date
   - Budget
   - Location
4. Click **"Create Project"**

### 5.5 Project Files

Each project can store related documents:
- Blueprints and drawings
- Contracts
- Permits
- Progress photos
- Reports

### 5.6 Bill of Materials (BOM)

| Quantity | Amount needed |
| Unit | Measurement unit |
| Unit Price | Cost per unit |
| Total | Calculated total cost |
| Supplier | Assigned supplier |
| Status | Procurement status |

### 6.3 Creating a Deal

1. Go to **Deals** in the sidebar
2. Click **"+ New Deal"**
3. Enter deal information:
   - Title (descriptive name)
   - Description
   - Select Supplier
   - Link to Project (optional)
   - Estimated Value
   - Probability (0-100%)
   - Expected Close Date
4. Click **"Create Deal"**

### 6.4 Deal Information

| Field | Description |
|-------|-------------|
| **Title** | Deal name/description |
| **Supplier** | Linked supplier |
| **Project** | Associated project |
| **Stage** | Current pipeline stage |
| **Estimated Value** | Expected deal value |
| **Quoted Value** | Actual quoted amount |
| **Final Value** | Agreed final amount |
| **Probability** | Likelihood of closing (%) |
| **Expected Close** | Target close date |

### 6.5 Updating Deal Stage

1. Open the deal
2. Select new stage from dropdown
3. Add notes about the change
4. Changes are saved automatically

   - Go to Activities
   - Log a Call or Email activity
   - Document what was discussed
   - Set follow-up date

   - Create a new deal
   - Change stage to "Quote Requested"

   - Enter quoted values

   - Negotiate terms
   - Update deal to "Negotiation" stage
   - When agreed, mark as "Won"

   - Update supplier rating
   - Add notes about experience
   - Mark as verified if appropriate

## 12. Frequently Asked Questions

### General Questions

**Q: Can I access Gold.Arch on my mobile device?**
A: Yes! A companion mobile app is available that syncs in real-time with the web application.

**Q: How many users can access the system?**
A: There is no limit on the number of users. Invite team members based on your subscription.

**Q: Is my data secure?**
A: Yes, all data is encrypted in transit and at rest. We use industry-standard security practices.

### Suppliers

**Q: How do I mark a supplier as verified?**
A: Verification status is typically managed by administrators after verifying supplier credentials.

**Q: Can I import suppliers from a spreadsheet?**
A: Import functionality is available for bulk uploads. Contact support for format requirements.

**Q: How is the supplier rating calculated?**
A: Ratings are based on owner ratings (manual input) and can be combined with external review data.

### Deals & Quotes

**Q: Can I link multiple quotes to one deal?**
A: Yes, you can receive and compare multiple quotes within a single deal.

**Q: What happens when I mark a deal as "Won"?**
A: The deal moves to completed status, and the final value is recorded for reporting.

**Q: Can I reopen a closed deal?**
A: Yes, deals can be reopened by changing their status back to an active stage.

### Tasks & Activities

**Q: Can I set recurring tasks?**
A: Recurring task functionality is planned for a future update.

**Q: How long are activity logs retained?**
A: Activity logs are retained indefinitely for auditing purposes.

### Login Issues

| Problem | Solution |
|---------|----------|
| Forgot password | Click "Forgot Password" on login page, check email for reset link |

### Data Issues

| Problem | Solution |
|---------|----------|
| Changes not saving | Check internet connection, refresh page, try again |
| Data not syncing | Refresh the page, clear browser cache |
| Missing suppliers | Check category filters, search in all categories |

### Performance Issues

| Problem | Solution |
|---------|----------|
| Slow loading | Clear browser cache, check internet speed |
| Page not responding | Refresh page, close unused browser tabs |
| Search not working | Wait for loading to complete, try simpler search terms |

| Error | Meaning | Action |
